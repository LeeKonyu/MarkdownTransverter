# 转换器操作方法

## 步骤

1. 该转换器需要您**提供**需要指定的**网址**

2. 所有您需要在桌面**创建**一个**名字为1**的文本文件，即**1.txt**

3. 在**1.txt**文件中**逐行**写下您需要将其转换的**网址**

   例如

   ```
   https://mp.weixin.qq.com/s/WG_JvCMQorpiqwh9O8AO0A
   https://mp.weixin.qq.com/s/YX0MGYD6S-HQSMrcimjhGg
   ```

   ![](https://focusteam-blog-pic.oss-cn-shenzhen.aliyuncs.com/20200312154106.png?OSSAccessKeyId=LTAI4FmmCn1teeyNYGo7r7px&Expires=1584002481&Signature=D1TKHvlsWjnte5tYYHqLGU%2BaNU8%3D)

4. 运行**trans.exe**文件，等待文件转换

5. 文件会在指**定目录**，即**D:\LLLLLLLLLLLLLLLLLLL\textTest**下生成md文件，生成的md文件会按照您输入的网址的**顺序**进行输出

   例如

   ```
   https://mp.weixin.qq.com/s/WG_JvCMQorpiqwh9O8AO0A，该网址对应text1
   https://mp.weixin.qq.com/s/YX0MGYD6S-HQSMrcimjhGg，该网址对应text2
   ```
